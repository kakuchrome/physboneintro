# リンクからインスタンスの入り方

一回インスタンス生成すれば、全員出た後でも、ワールド情報さえあればインスタンスを維持できる（インスタンス主・インスタンスタイプの維持）  

## フレンド・フレンドプラのインスタンスタイプ別
Lockedインスタンス：リンクからインスタンス主であればフレンドでなければ入れない  
Unlockedインスタンス：リンクをしていれば誰でも入れるインスタンス
